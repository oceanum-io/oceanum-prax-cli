Oceanum PRAX - CLI
==================

The Oceanum PRAX CLI is a command-line utility to interact with the Oceanum PRAX platform.

The Oceanum PRAX platform is a Cloud-based service for building and deploying front-end Apps, back-end services and data-pipelines workflows to the Oceanum infrastructure.

A PRAX Project can be defined using a single YAML file that describes the project structure, secrets, tasks, pipelines, apps and other resouces.

.. note::
    Oceanum PRAX platform is a Beta feature and it's available by invitation only. 
    
    Please contact us for more information at support@oceanum.io!
